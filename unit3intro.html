Unit 3: Numerical Methods

This section explores methods that help us find solutions when exact answers are difficult or impossible to calculate.

Key Concepts:

Interpolation:

  • Lagrange Interpolation: A method to estimate unknown values between known data points by creating a polynomial that fits those points.
  • Inverse Interpolation: Used to find the value of an independent variable (like x) when the dependent variable (like y) is given.

Finite Differences:

  • Forward Differences: A technique to calculate the difference between successive data points.
  • Backward Differences: Similar to forward differences but works in the opposite direction.
  • Divided Differences: A method to handle unevenly spaced data points by calculating differences between them.

Difference Tables:

  • Forward and Backward Difference Tables: Tables used to organize and simplify the process of calculating differences for interpolation.
  • Divided Difference Table: A table that organizes divided differences, which helps in interpolation when data points are not equally spaced.

Newton's Methods of Interpolation:

  • Newton's Forward Difference Method: A formula used to estimate values when data points are evenly spaced.
  • Newton's Backward Difference Method: Similar to the forward method but used when values at the end of the data set are more important.
  • Newton's Divided Difference Method: This method is used for unevenly spaced data points and gives a polynomial for interpolation.

Numerical Integration:

  • Newton-Cotes Integration Formula: A general way to approximate the area under a curve, which is useful for integration.
  • Trapezoidal Rule: A simple method to estimate the area under a curve by approximating it as a series of trapezoids.
